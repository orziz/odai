# AISkills

本 Repo 主要是用于一些基本的AI编程相关的辅助，提升趣味性及优化流程

## 如何使用

### claude

放进同名文件夹下即可

#### commands
由 claude 的指令触发，输入框输入 `/指令名` 即可，如 `/sslb`

### copilot

将 `.github/skills` 下面的内容，酌情复制到 `.github/copilot-instructions.md` 里即可
> 因为copilot会自行读 `.github/copilot-instructions.md` 所以无需其他操作
