# FX Cinematics Playbook

## 何时读取

命中以下任一主题时，读本文件：

- 技能特效、命中特效、受击反馈、Buff / Debuff 状态表现
- 镜头演出、转场包装、胜负结算和关键剧情演出
- 战斗读招清晰度、视觉节奏和信息干扰
- 性能压力、降级方案和多人同屏可读性边界

## 最小顺序

1. 先定反馈任务：这段表现是为了读招、强调强度、制造爽感，还是承接剧情情绪。
2. 再定层级：判定信息、伤害信息、特效主体和镜头运动谁优先。
3. 再定约束：性能预算、同屏密度、降级方案和低端机边界。
4. 末补验收：清晰度、节奏、疲劳度与性能是否成立。

## 特效短框架

至少补齐：技能前摇/命中/收招/残留的节奏分层；强弱技能、友军/敌军、不同伤害类型识别性；特效是否遮挡判定、资源、血量、冷却或引导信息。

## 镜头与演出短框架

至少补齐：镜头运动、转场节奏、信息出现时机；演出是否抢走玩家控制/判断的关键信息；长看或高频触发后是否疲劳、晕眩或节奏噪音。

## 性能与降级

至少补齐：多人同屏、Boss 高压、弱网或低端机下可读性边界；可关闭/可简化/必须保留的表现层；降级后是否仍保核心判定与反馈。

## 最小交付

按主矛盾选：

- 特效方向：反馈目标、层级、清晰度、性能和风险
- 镜头 / 演出方向：情绪目标、节奏、信息优先级和疲劳边界
- 性能 / 降级规则：同屏密度、保留层、简化层和最低可读标准
