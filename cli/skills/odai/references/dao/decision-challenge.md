# 高代价决策的独立挑战与合议征询

## 何时读取

当前决定高代价或难回退，待冻结结论将约束跨模块主路、核心逻辑、公开契约、数据、权限或发布，或正式审查结论将作合并、发布、安全、全仓 / 全目录准入门时读取。本文件只治理待冻结结论的独立挑战与合议升档；实际调用 agent / 模型、并行或执行下放时另读 `references/dao/agent-governance.md`。

## 自动一席挑战

命中读取条件时，最终冻结前必须做 1 席独立挑战；不允许主流程自评后直接冻结，也不再另做成本收益门判断。

挑战席围绕同一已锁定题面与待冻结结论找反例、漏判、备路与不可接受结果，不能复述主流程。挑战输入只含已锁定题面、范围、待冻结结论和完整正反证据，不得暗示预期结论、要求忽略某类问题或隐藏反面材料。主流程须收口为采纳、部分采纳或驳回并说明理由。

宿主有异构模型时优先异构；只有同系 agent 时用同模型独立挑战且切入点必须不同；无 agent 能力时由主流程显式反证。实际调用 agent / 模型前先读 `references/dao/agent-governance.md`，不得伪称调用。

## 合议升档

用户显式说 `增强模式` / `合议` / `多模型复核` / `开合议` 或等价表达时，读取 `references/dao/consensus-mode.md` 进入 N 席重型版；否则先完成自动 1 席挑战及主流程收口。

未显式激活时，若一席挑战后决策仍同时满足高 stakes、真多路、难回退，且一席不足以覆盖反例、备路和不可接受结果，征询用户是否升为 N 席合议；用户同意才读取 `references/dao/consensus-mode.md`，拒绝则沿用 1 席收口。同一独立决策点在本次请求内只征询一次；用户拒绝后，题面、风险、备选路线或宿主能力无新证据时不得复提。
